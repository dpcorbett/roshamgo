# Roshamgo

This repository contains a web-based game similar to "Rock, Paper, Scissors," but expanded to include five choices: "rock", "paper", "scissors", "rope", and "fire".

## How to Play

Players select their move by clicking on one of the image icons. The computer then randomly makes its own choice, and the game determines the winner based on a set of predefined rules for the five elements. The outcome of the round is displayed directly on the page in a dedicated message area.

## Technology Stack

*   **HTML:** For the page structure and game interface.
*   **CSS:** Utilises Bootstrap for responsive layout and `css/main.css` for custom styling.
*   **JavaScript:** `js/script.js` contains the core game logic, including choice generation and comparison.
*   **jQuery:** Used for dynamic effects on page load and interaction with the game result display.
